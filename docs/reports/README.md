# Generated Validation Reports

This directory is the standard location for Make3D portfolio validation artifacts generated by CI or local dry-runs.

Expected generated files:

```text
portfolio_validation_report.md
portfolio_validation_report.json
make3d_generated_sample_manifest.json
```

The workflow also uploads generated sample artifacts from the CMake build directory, including production pipeline outputs and generated portfolio samples.

Local command:

```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
ctest --test-dir build -C Release --output-on-failure
python scripts/portfolio_validation.py --dry-run --report-dir docs/reports --build-dir build
```

Rollback is simple because generated files are isolated under the build directory and report directory.
